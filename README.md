# ❤️ AI-Enhanced Arrhythmia Detection

## 📌 Overview

This project uses machine learning and deep learning models to detect cardiac arrhythmia from ECG data.
It helps in early identification of heart abnormalities using automated analysis.

---

## ⚙️ Tech Stack

* Python
* NumPy, Pandas
* Scikit-learn
* Matplotlib

---

## 🧠 Models Used

* CNN (best performance)
* Random Forest
* SVM
* XGBoost
* AdaBoost

---

## 📊 Results

* CNN achieved highest accuracy (~89%)
* Random Forest provided good balance between speed and accuracy

---

## 🎯 Key Idea

The model analyzes ECG data and classifies whether the signal indicates a normal or abnormal heart condition.
